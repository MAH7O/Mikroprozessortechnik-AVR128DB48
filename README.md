# Mikroprozessortechnik (MPT) - AVR128DB48 Projects

This repository contains a collection of exercises and projects developed during the **Mikroprozessortechnik** module. The work covers fundamental concepts of microcontroller programming, ranging from low-level Assembly to high-level C implementations using the **AVR128DB48** microcontroller.

## 🚀 Overview
The projects demonstrate various aspects of embedded systems development, including hardware interfacing, peripheral control, and signal processing.

## 📂 Repository Structure
The exercises are organized chronologically and by topic:

1.  **[01-Assembly-Basics](./01-Assembly-Basics)**: Introduction to Assembly language, basic instructions, and output handling.
2.  **[02-C-Programming](./02-C-Programming)**: Transitioning to C for microcontroller applications.
3.  **[03-Sensor-Application](./03-Sensor-Application)**: Implementing sensor logic and data acquisition.
4.  **[04-GPIO-LED-Control](./04-GPIO-LED-Control)**: General Purpose Input/Output (GPIO) operations and LED patterns.
5.  **[05-I2C-LCD-Interface](./05-I2C-LCD-Interface)**: Interfacing with LCD displays using the I2C protocol.
6.  **[06-Interrupts-Timers](./06-Interrupts-Timers)**: Handling hardware interrupts and utilizing timers for precision tasks.
7.  **[07-PWM-Servo-Control](./07-PWM-Servo-Control)**: Using Pulse Width Modulation (PWM) to control servo motors.
8.  **[08-ADC-Conversion](./08-ADC-Conversion)**: Analog-to-Digital Conversion (ADC) for reading sensor values.
9.  **[09-I2C-Advanced](./09-I2C-Advanced)**: Advanced I2C communication and complex peripheral interfacing.
10. **[10-DAC-Bonus-Project](./10-DAC-Bonus-Project)**: A bonus project involving Digital-to-Analog Conversion (DAC) to synthesize audio/songs.

## 🛠️ Hardware & Tools
- **Microcontroller**: AVR128DB48
- **Programming Languages**: Assembly, C
- **IDE/Tools**: Microchip Studio, VS Code, PlatformIO (as applicable)
